# `openclaw setup`

`~/.openclaw/openclaw.json` နှင့် အေးဂျင့် workspace ကို စတင်အတည်ပြုပါ။

ဆက်စပ်အကြောင်းအရာများ—

- စတင်ရန်: [စတင်ရန်](/start/getting-started)
- လမ်းညွှန်ကိရိယာ: [စတင်အသုံးပြုခြင်း](/start/onboarding)

## ဥပမာများ

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

setup မှတဆင့် wizard ကို အလုပ်လုပ်စေရန်—

```bash
openclaw setup --wizard
```
