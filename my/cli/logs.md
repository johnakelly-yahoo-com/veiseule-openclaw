# `openclaw logs`

RPC မှတစ်ဆင့် Gateway（ဂိတ်ဝေး） ဖိုင်လော့ဂ်များကို tail လုပ်ပါ (remote mode တွင် အလုပ်လုပ်ပါသည်)။

ဆက်စပ်အကြောင်းအရာများ:

- လော့ဂ်ခြင်း အနှစ်ချုပ်: [လော့ဂ်ခြင်း](/logging)

## ဥပမာများ

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
