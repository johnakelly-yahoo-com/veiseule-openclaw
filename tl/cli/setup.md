# `openclaw setup`

I-initialize ang `~/.openclaw/openclaw.json` at ang agent workspace.

Kaugnay:

- Pagsisimula: [Pagsisimula](/start/getting-started)
- Wizard: [Pagsisimula](/start/onboarding)

## Mga halimbawa

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Para patakbuhin ang wizard sa pamamagitan ng setup:

```bash
openclaw setup --wizard
```
