# `openclaw skills`

Siyasatin ang mga Skills (bundled + workspace + managed overrides) at tingnan kung alin ang karapat-dapat kumpara sa mga kulang na kinakailangan.

Kaugnay:

- Sistema ng Skills: [Skills](/tools/skills)
- Config ng Skills: [Config ng Skills](/tools/skills-config)
- Mga install ng ClawHub: [ClawHub](/tools/clawhub)

## Mga command

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
