# `openclaw logs`

I-tail ang mga file log ng Gateway sa pamamagitan ng RPC (gumagana sa remote mode).

Kaugnay:

- Pangkalahatang-ideya ng Logging: [Logging](/logging)

## Mga halimbawa

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
