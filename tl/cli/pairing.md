# `openclaw pairing`

Aprubahan o siyasatin ang mga DM pairing request (para sa mga channel na sumusuporta sa pairing).

Kaugnay:

- Daloy ng Pairing: [Pairing](/channels/pairing)

## Mga command

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
