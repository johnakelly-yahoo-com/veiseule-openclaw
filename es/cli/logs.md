# `openclaw logs`

Siga en tiempo real los registros de archivos del Gateway mediante RPC (funciona en modo remoto).

Relacionado:

- Descripción general de registros: [Logging](/logging)

## Ejemplos

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
