# TOOLS.md - 使用者工具備註（可編輯）

This file is for _your_ notes about external tools and conventions.
此檔案用於記錄「你」關於外部工具與慣例的備註。
它不定義存在哪些工具；OpenClaw 會在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage／SMS：描述對象與內容，並在傳送前確認。
- 偏好簡短訊息；避免傳送機密資訊。

### sag

- 文字轉語音：指定語音、目標說話者／房間，以及是否要串流。

加入任何你希望助理了解的本機工具鏈相關資訊。
