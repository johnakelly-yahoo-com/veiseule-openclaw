# `openclaw pairing`

Phê duyệt hoặc kiểm tra các yêu cầu ghép đôi DM (đối với các kênh hỗ trợ ghép đôi).

Liên quan:

- Luồng ghép đôi: [Pairing](/channels/pairing)

## Các lệnh

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
