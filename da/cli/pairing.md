# `openclaw pairing`

Godkend eller inspicér parringsanmodninger for DM’er (for kanaler, der understøtter parring).

Relateret:

- Parringsforløb: [Parring](/channels/pairing)

## Kommandoer

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
