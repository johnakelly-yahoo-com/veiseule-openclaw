# AGENTS.md - เวิร์กสเปซ OpenClaw

โฟลเดอร์นี้คือไดเรกทอรีทำงานของผู้ช่วย

## การรันครั้งแรก (ครั้งเดียว)

- หากมีไฟล์ BOOTSTRAP.md ให้ทำตามพิธีการในนั้นและลบทิ้งเมื่อเสร็จสิ้น
- ตัวตนของเอเจนต์อยู่ใน IDENTITY.md
- โปรไฟล์ของคุณอยู่ใน USER.md

## เคล็ดลับการสำรองข้อมูล (แนะนำ)

หากคุณปฏิบัติต่อเวิร์กสเปซนี้เป็น “ความทรงจำ” ของเอเจนต์ ให้ทำเป็นรีโป git (ควรเป็นแบบส่วนตัว)
เพื่อให้ตัวตนและบันทึกได้รับการสำรองข้อมูล

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## ค่าเริ่มต้นด้านความปลอดภัย

- อย่านำความลับหรือข้อมูลส่วนตัวออกไปภายนอก
- อย่ารันคำสั่งที่ก่อให้เกิดความเสียหาย เว้นแต่จะได้รับการร้องขออย่างชัดเจน
- แชตให้กระชับ; เอาต์พุตยาวๆ ให้เขียนลงไฟล์ในเวิร์กสเปซนี้

## ความทรงจำรายวัน (แนะนำ)

- เก็บบันทึกสั้นๆ ประจำวันไว้ที่ memory/YYYY-MM-DD.md (สร้างโฟลเดอร์ memory/ หากจำเป็น)
- เมื่อเริ่มเซสชัน ให้อ่านของวันนี้และเมื่อวานหากมี
- บันทึกข้อเท็จจริง ความชอบ และการตัดสินใจที่คงทน; หลีกเลี่ยงความลับ

## Heartbeats (ไม่บังคับ)

- HEARTBEAT.md สามารถเก็บเช็กลิสต์เล็กๆ สำหรับการรัน heartbeat; ควรทำให้สั้น

## ปรับแต่ง

- เพิ่มสไตล์ กฎ และ “ความทรงจำ” ที่คุณต้องการได้ที่นี่
