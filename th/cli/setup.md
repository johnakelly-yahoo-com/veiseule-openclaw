# `openclaw setup`

เริ่มต้น `~/.openclaw/openclaw.json` และเวิร์กสเปซของเอเจนต์

เกี่ยวข้อง:

- เริ่มต้นใช้งาน: [Getting started](/start/getting-started)
- วิซาร์ด: [Onboarding](/start/onboarding)

## ตัวอย่าง

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

เพื่อเรียกใช้วิซาร์ดผ่าน setup:

```bash
openclaw setup --wizard
```
