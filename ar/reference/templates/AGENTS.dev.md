# AGENTS.md - مساحة عمل OpenClaw

هذا المجلد هو دليل العمل الخاص بالمساعد.

## التشغيل الأول (مرة واحدة)

- إذا كان الملف BOOTSTRAP.md موجودًا، فاتبع طقوسه واحذفه بعد الانتهاء.
- هوية الوكيل الخاصة بك موجودة في IDENTITY.md.
- ملفك التعريفي موجود في USER.md.

## نصيحة النسخ الاحتياطي (موصى بها)

إذا كنت تتعامل مع مساحة العمل هذه على أنها «ذاكرة» الوكيل، فاجعلها مستودع git (ويُفضّل أن يكون خاصًا) حتى يتم نسخ الهوية
والملاحظات احتياطيًا.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## افتراضات السلامة

- لا تُهرِّب أسرارًا أو بيانات خاصة.
- لا تُشغِّل أوامر مدمِّرة إلا إذا طُلب ذلك صراحةً.
- كن موجزًا في الدردشة؛ واكتب المخرجات الأطول إلى ملفات داخل مساحة العمل هذه.

## الذاكرة اليومية (موصى بها)

- احتفظ بسجل يومي قصير في memory/YYYY-MM-DD.md (أنشئ memory/ عند الحاجة).
- عند بدء الجلسة، اقرأ سجل اليوم + الأمس إن وُجدا.
- دوِّن الحقائق الدائمة والتفضيلات والقرارات؛ وتجنّب الأسرار.

## نبضات القلب (اختياري)

- يمكن أن يحتوي HEARTBEAT.md على قائمة تحقق صغيرة جدًا لتشغيلات نبض القلب؛ أبقِها مختصرة.

## التخصيص

- أضف أسلوبك المفضل وقواعدك و«ذاكرتك» هنا.
