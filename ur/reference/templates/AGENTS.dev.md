- اگر BOOTSTRAP.md موجود ہو تو اس کے طریقۂ کار پر عمل کریں اور مکمل ہونے کے بعد اسے حذف کر دیں۔

## بیک اپ مشورہ (سفارش کردہ)

اگر آپ اس ورک اسپیس کو ایجنٹ کی "یادداشت" سمجھتے ہیں تو اسے ایک git repo بنا لیں (ترجیحاً نجی)، تاکہ شناخت
اور نوٹس کا بیک اپ محفوظ رہے۔

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```
