# `openclaw skills`

Skills (بنڈل شدہ + ورک اسپیس + منیجڈ اووررائیڈز) کا معائنہ کریں اور دیکھیں کہ کون سی اہل ہیں بمقابلہ کن تقاضوں کی کمی ہے۔

متعلقہ:

- Skills سسٹم: [Skills](/tools/skills)
- Skills کنفیگ: [Skills config](/tools/skills-config)
- ClawHub انسٹالز: [ClawHub](/tools/clawhub)

## کمانڈز

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
