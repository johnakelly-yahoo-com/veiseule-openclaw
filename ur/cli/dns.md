# `openclaw dns`

وسیع علاقے کی دریافت کے لیے DNS معاون ٹولز (Tailscale + CoreDNS)۔ فی الحال macOS + Homebrew CoreDNS پر مرکوز ہے۔

متعلقہ:

- Gateway ڈسکوری: [Discovery](/gateway/discovery)
- وسیع علاقائی ڈسکوری کنفیگ: [Configuration](/gateway/configuration)

## سیٹ اپ

```bash
openclaw dns setup
openclaw dns setup --apply
```
