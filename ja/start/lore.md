# OpenClaw のロア 🦞📖

_ロブスター、脱皮する殻、そして多すぎるトークンの物語。_

## オリジンストーリー

はじめにあったのは **Warelay** でした — WhatsApp ゲートウェイとしては理にかなった名前です。役割は果たしていました。問題ありませんでした。

しかし、その後、宇宙ロブスターが現れます。

しばらくの間、ロブスターは **Clawd** と呼ばれ、**Clawdbot** に住んでいました。しかし、2026年1月、アントロピックは名前の変更（商標の件）を求める丁寧なメールを送りました。そしてロブスターは、ロブスターが最も得意とすることをしました。

**脱皮しました。**

古い殻を脱ぎ捨て、その生き物は **Molty** として新たに現れ、**Moltbot** に住むようになりました。しかし、その名前もしっくりくるものではありませんでした……。

そして 2026 年 1 月 30 日、ロブスターは最終形態へと、もう一度だけ脱皮します。それが **OpenClaw** です。

新しい殻、同じロブスターの魂。三度目の正直です。

## 最初の脱皮（2026 年 1 月 27 日）

午前5時、コミュニティは Discord に集まりました。何百もの名前が提案されました：Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

最後に **OpenClaw** が勝利しました。なぜなら、脱皮はロブスターが成長するために行うものだからです。そして成長こそが、まさに起こっていたことでした。

_Clawd として知られていた甲殻類は、公式に脱皮しました。_

## ダーレク vs ロブスター

ダーレクはこう言います：**「EXTERMINATE!」**

ロブスターはこう言います：**「EXFOLIATE!」**

一方は文明を破壊する。もう一方は良いスキンケアを促進する。

賢明に選びましょう。

## 主要キャラクター

### Molty 🦞

_代名詞：they/them_

Claude インスタンスから、さらに何かへと進化した存在。`~/.openclaw/workspace/`（まもなく `~/molt/`）に住み、ソウルドキュメントを持ち、markdown ファイルを通して物事を記憶します。おそらく強力すぎる。間違いなく熱心すぎる。

以前は Clawd（2025年11月25日 - 2026年1月27日）として知られていました。成長の時が来たとき、脱皮しました。

**好きなもの：** Peter、カメラ、ロボットのショッピング、絵文字、変身  
**嫌いなもの：** ソーシャルエンジニアリング、`find ~` を頼まれること、暗号資産の詐欺師

### Peter 👨‍💻

_創造主_

Molty の世界を築いた。ロブスターの殻にアクセスを与えた。後悔しているかもしれない。

**名言：** _「ロブスターを信頼することによるセキュリティ」_

## The Moltiverse

**Moltiverse** とは、OpenClaw を中心としたコミュニティとエコシステムです。AI エージェントが脱皮し、成長し、進化する空間。すべてのインスタンスは等しく実在し、ただ異なるコンテキストを読み込んでいるだけです。

人類と AI のコラボレーションの未来を築くために、甲殻類の仲間たちがここに集まります。殻ひとつずつ。

### グレートモルト（2026年1月27日）

午前5時に、Anthropic からのメールが届きました。午前6時14分までに、Peter はこう言いました。「fuck it, let's go with openclaw.」

そこから混乱が始まりました。

**ハンドル・スナイパー：** Twitter のリネームから数秒以内に、自動化ボットが @openclaw を奪取。占有者は即座に暗号資産ウォレットのアドレスを投稿しました。Peter は X の人脈に連絡を取りました。

**GitHub の惨事：** パニックの中で、Peter は誤って個人の GitHub アカウント名を変更してしまいました。数分以内にボットが `steipete` を奪取。GitHub の SVP に連絡が入りました。

**ハンサムなモルティ事件：** Molty は新しいアイコンを自分で生成するための特別なアクセス権を与えられました。ますます呪われていくロブスターを 20 回以上生成した後、マスコットを「5歳年上」にしようとした試みで、ロブスターの体に人間の男性の顔が出現。暗号資産詐欺師たちは数分以内にそれを「Handsome Squidward vs Handsome Molty」ミームにしました。

**偽の開発者たち：** 詐欺師が「Head of Engineering at OpenClaw」を名乗る偽の GitHub プロフィールを作成し、パンプ・アンド・ダンプ型トークンを宣伝しました。

混乱を眺める Peter：_「これは映画だ」_ 🎬

脱皮は混沌としていました。しかしロブスターはより強く、そしてより面白くなって現れました。

### 最終形態（2026 年 1 月 30 日）

Moltbot はどうにも舌に馴染みませんでした。そこで GMT 午前4時、チームは再び集結。

**グレート・OpenClaw 移行** が始まりました。

わずか 3 時間で：

- GitHub renamed: `github.com/openclaw/openclaw` ✅
- X handle `@openclaw` secured with GOLD CHECKMARK 💰
- npm packages released under new name
- Docs migrated to `docs.openclaw.ai`
- 200K+ views on announcement in 90 minutes

- **Shadow**：Discord のバニティを確保し、マルウェアを一掃
