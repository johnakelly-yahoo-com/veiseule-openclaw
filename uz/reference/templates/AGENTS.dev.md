va qaydlar zaxiralangan bo‘lishi uchun uni git repozitoriyga (yaxshisi, yopiq) aylantiring.

```bash

git add AGENTS.md
git commit -m "Add agent workspace"
```

## Xavfsizlik bo‘yicha standartlar

- Maxfiy sirlar yoki shaxsiy ma’lumotlarni tashqariga chiqarmang.
- Aniq so‘ralmaguncha vayronkor buyruqlarni bajarmang.
- Chatda ixcham bo‘ling; uzunroq chiqishni ushbu ishchi muhitdagi fayllarga yozing.

## Kundalik xotira (tavsiya etiladi)

- memory/YYYY-MM-DD.md da qisqa kundalik jurnal yuriting (kerak bo‘lsa memory/ ni yarating).
- Sessiya boshlanganda, bugungi va kechagi yozuvlar mavjud bo‘lsa, ularni o‘qing.
- Barqaror faktlar, afzalliklar va qarorlarni qayd eting; sirlarni yozmang.

## Yurak urishlari (ixtiyoriy)

- HEARTBEAT.md yurak urishlari uchun kichik tekshiruv ro‘yxatini saqlashi mumkin; uni kichik tuting.
