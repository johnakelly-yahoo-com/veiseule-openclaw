# TOOLS.md — Mahalliy qaydlar

Ko‘nikmalar asboblar _qanday_ ishlashini belgilaydi. This file is for _your_ specifics — the stuff that's unique to your setup.

## Bu yerga nimalar kiritiladi

Masalan:

- Camera names and locations
- SSH xostlari va taxalluslar
- TTS uchun afzal ko‘rilgan ovozlar
- Karnay/xona nomlari
- Qurilma laqablari
- Muhitga xos bo‘lgan har qanday narsa

## Misollar

```markdown
### Kameralar

- living-room → Asosiy hudud, 180° keng burchak
- front-door → Kirish, harakat bilan ishga tushadi

### SSH

- home-server → 192.168.1.100, foydalanuvchi: admin

### TTS

- Afzal ovoz: "Nova" (iliq, biroz britancha)
- Standart karnay: Kitchen HomePod
```

## Nega ajratish kerak?

Ko‘nikmalar umumiy. Sizning sozlamangiz — faqat sizniki. Ularni alohida saqlash ko‘nikmalarni qaydlaringizni yo‘qotmasdan yangilashga, hamda infratuzilmangizni oshkor qilmasdan ko‘nikmalarni ulashishga imkon beradi.
