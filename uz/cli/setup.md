`~/.openclaw/openclaw.json` faylini va agent workspace’ini ishga tushiradi.

Bog‘liq:

- Boshlash: [Getting started](/start/getting-started)

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```
