# `openclaw reset`

Mahalliy konfiguratsiya/holatni tiklash (CLI o‘rnatilgan holda qoladi).

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
