# `openclaw logs`

Gateway fayl loglarini RPC orqali real vaqtda ko‘rish (masofaviy rejimda ishlaydi).

Bog‘liq:

- Loglash bo‘yicha umumiy ma’lumot: [Loglash](/logging)

## Misollar

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
