# `openclaw tui`

Gateway’ga ulangan terminal UI’ni ochadi.

Bog‘liq:

- TUI qo‘llanmasi: [TUI](/web/tui)

## Misollar

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
