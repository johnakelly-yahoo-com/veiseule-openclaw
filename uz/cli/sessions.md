# `openclaw skills` (list/info/check) va skill mosligi uchun CLI ma’lumotnomasi

Saqlangan suhbat sessiyalarini ro‘yxatlash.

```bash
openclaw sessions
openclaw sessions --active 120
openclaw sessions --json
```
