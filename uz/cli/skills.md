# `--deep` jonli tekshiruvlarni ishga tushiradi (WhatsApp Web + Telegram + Discord + Google Chat + Slack + Signal).

Ko‘nikmalarni (biriktirilgan + ish maydoni + boshqariladigan override’lar) ko‘zdan kechiring va qaysilari mos kelishini hamda qaysilarida talablar yetishmasligini ko‘ring.

Bog‘liq:

- Ko‘nikmalar tizimi: [Skills](/tools/skills)
- Ko‘nikmalar sozlamalari: [Skills config](/tools/skills-config)
- ClawHub o‘rnatishlari: [ClawHub](/tools/clawhub)

## Buyruqlar

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
