# `openclaw pairing`

Approuver ou inspecter les demandes d'appairage de DM (pour les canaux qui supportent l'appairage).

Liens connexes :

- Flux d’appariement : [Appariement](/start/pairing)

## Commandes

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
