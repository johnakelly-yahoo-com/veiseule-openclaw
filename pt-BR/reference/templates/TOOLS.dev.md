# TOOLS.md - Notas de ferramentas do usuário (editável)

Este arquivo é para _suas_ notas sobre ferramentas externas e convenções.
Ele não define quais ferramentas existem; o OpenClaw fornece ferramentas integradas internamente.

## Exemplos

### imsg

- Enviar um iMessage/SMS: descreva quem/o quê, confirme antes de enviar.
- Prefira mensagens curtas; evite enviar segredos.

### sag

- Texto para fala: especifique a voz, o alto-falante/sala de destino e se deve transmitir.

Adicione qualquer outra coisa que você queira que o assistente saiba sobre sua toolchain local.
