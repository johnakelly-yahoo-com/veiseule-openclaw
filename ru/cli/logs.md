# `openclaw logs`

Следить в реальном времени за файловыми логами Gateway (шлюза) по RPC (работает в удалённом режиме).

Связанное:

- Обзор логирования: [Logging](/logging)

## Примеры

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
