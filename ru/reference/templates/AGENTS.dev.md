# AGENTS.md — Рабочее пространство OpenClaw

Эта папка является рабочим каталогом ассистента.

## Первый запуск (один раз)

- Если существует BOOTSTRAP.md, следуйте его ритуалу и удалите файл после завершения.
- Идентичность вашего агента хранится в IDENTITY.md.
- Ваш профиль хранится в USER.md.

## Совет по резервному копированию (рекомендуется)

Если вы рассматриваете это рабочее пространство как «память» агента, сделайте его git-репозиторием
(желательно приватным), чтобы идентичность и заметки были сохранены в резервной копии.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## Настройки безопасности по умолчанию

- Не выводите секреты или приватные данные.
- Не выполняйте разрушительные команды, если это явно не запрошено.
- В чате будьте кратки; более длинный вывод записывайте в файлы в этом рабочем пространстве.

## Ежедневная память (рекомендуется)

- Ведите короткий ежедневный журнал в memory/YYYY-MM-DD.md (при необходимости создайте memory/).
- При начале сеанса прочитайте сегодняшний и вчерашний файлы, если они существуют.
- Фиксируйте устойчивые факты, предпочтения и решения; избегайте секретов.

## Heartbeats (необязательно)

- HEARTBEAT.md может содержать небольшой чек-лист для запусков heartbeat; держите его компактным.

## Настроить

- Добавьте здесь предпочитаемый стиль, правила и «память».
