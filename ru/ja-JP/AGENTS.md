# AGENTS.md — рабочая область перевода документации ja-JP

## Читать, когда

- Поддержка `docs/ja-JP/**`
- Обновление конвейера перевода на японский (глоссарий/TM/prompt)
- Обработка отзывов или регрессий японского перевода

## Конвейер (docs-i18n)

- Исходная документация: `docs/**/*.md`
- Целевая документация: `docs/ja-JP/**/*.md`
- Глоссарий: `docs/.i18n/glossary.ja-JP.json`
- Память переводов: `docs/.i18n/ja-JP.tm.jsonl`
- Правила prompt: `scripts/docs-i18n/prompt.go`

Часто используемые команды:

```bash
# Массово (режим doc; параллельный запуск допустим)
cd scripts/docs-i18n
go run . -docs ../../docs -lang ja-JP -mode doc -parallel 6 ../../docs/**/*.md

# Один файл
cd scripts/docs-i18n
go run . -docs ../../docs -lang ja-JP -mode doc ../../docs/start/getting-started.md

# Небольшие правки (режим segment; использует TM; без параллелизма)
cd scripts/docs-i18n
go run . -docs ../../docs -lang ja-JP -mode segment ../../docs/start/getting-started.md
```

Примечания:

- Предпочитайте режим `doc` для перевода всей страницы; режим `segment` — для небольших исправлений.
- Если очень большой файл вызывает тайм-аут, выполните точечные правки или разделите страницу перед повторным запуском.
- После перевода выполните выборочную проверку: фрагменты/блоки кода не изменены, ссылки/якоря не изменены, плейсхолдеры сохранены.
