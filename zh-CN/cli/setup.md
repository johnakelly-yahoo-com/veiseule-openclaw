# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和智能体工作区。

相关内容：

- 快速开始：[快速开始](/start/getting-started)
- 向导：[新手引导](/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```
